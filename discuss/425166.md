 <br/>
 <br/>
 两个不开摄像头的男面试官。
 <br/>
 <br/>
 了解了一些基本情况，然后进行自我介绍。针对自我介绍和项目经历问一些基础问题。比如我说会C和Java，问这两种语言的区别；说毕设是NLP，问NLP用到的神经网络。
 <br/>
 <br/>
 总之要熟悉自己的简历，对自己说出来的内容负责。
 <br/>
 <br/>
 面试官很nice，氛围比较轻松，我表现得不太好也不觉得慌。
 <br/>
 <br/>
 反问二面通知时间，答下周陆续通知。又问二面的组织形式，答线下概率更大。

 15、父进程退出了，子进程会不会退出？
 <br/>
 <br/>
 ```
 <br/>
 若父进程退出，子进程尚未结束，则子进程会被init进程领养，也就是说init进程将成为该子进程的父进程。
 <br/>
 <br/>
 若希望父进程退出，子进程也退出的话，可以使用线程，因为若进程结束，则还没结束的线程一定会立刻结束。或者用信号，用atexit注册一个退出时执行的函数，这个函数中发送信号给子进程即可。
 <br/>
 ```
